## docsify

官网： https://docsify.js.org/#/  
源码：https://github.com/docsifyjs/docsify  

> 依赖 node.js 环境

### 安装<small>(全局)</small>

> `npm i docsify-cli -g`

### 创建

> `docsify init ./docs`

### 启动

> `docsify serve ./docs -p 80 --open`
